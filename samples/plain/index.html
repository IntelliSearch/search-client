IntelliSearch Search-Client | Plain sample

This sample page demonstrates how to set up a connection to the IntelliSearch search-engine using the automated strategy.
The automated strategy means that you use the settings object to tell how you want the engine to handle the various scenarios, and based on that the autocomplete suggestions, categories and match-results will be sent to you via three callback methods, when there has been any activity. You then set up events to forward changes to the search queryfield and other buttons (paging, search-type, etc).
